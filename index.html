HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The section element defines a section in a documenet that contains thematic content
  • Section elements go inside the main element
  • h1, h2, h3, etc. define header tiers
  • ul tags create an unordered list of items, ol tags create ordered lists, and li tags create list items
  • The `src` attribute defines the location, or source, of the image file.
  • The `alt` attribute contains a text string that describes the appearance and functionality of the image.

CSS

  • There are three ways to style a webpage using CSS: inline CSS, internal CSS style sheet, and external CSS style sheet.
  • Inline CSS is used to style an element directly in HTML.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
